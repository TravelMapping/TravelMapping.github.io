Travel Mapping

This site is maintained as a place where users and contributors to the Travel Mapping project can find information in the event that the project's main server is offline.

Project Status

The the travelmapping.net domain is currently inaccessible to many due to a DNS configuration problem. The problem has been identified and will be fixed soon. It might take some time for the fix to propagate to name servers that have cached the problematic information. For now, you should be able to access the staging site and have access to most main site functionality.
